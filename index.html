Out-of-Order RISC-V Processor

Project 1 Detailed Image

Overview

  • 8-Week course project (EECS 470) to build an Out-of-Order processor with Speculative Execution and additional advanced features.
  • Programmed in SystemVerilog and Fully Synthesizable.
  • Rigorously tested modules using unit tests and processor execution using programs such as AlexNet and a Simulated CPU in C.
  • Average CPI: 1.3
  • Clock Period: 11.5ns (86.95 MHz)

Advanced Features

  • N-Way Superscalar
  • Store Queue (with Store to Load Forwarding)
  • Early Tag Broadcast
  • Instruction Prefetching
  • GShare Branch Predictor
